What did i learn today?

firstly i learned an easy way to test the responsiveness of my website

by using the inspect tool on a live server of my page i could toggle devices or manually set the width and height

i noticed that below a certain width my Nav bar would start to overlap and become distorted

i raised this in a buddy meeting as an issue i was working on and was directed to media querys

I used this site for reference on screen sizes

the code i created was this:


@media (max-width: 479px) {
nav ul li {
display: block; }
}

this would convert my display from inline to block so that the buttons stack on top of each other on a smaller screen width
