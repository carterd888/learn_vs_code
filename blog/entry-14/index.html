What did i learn today?

today i wanted to improve my nav bar using a flexbox

to do this i created a branch from my master so could save and change things freely

i then made the unordered list display:flex this is the parent container for the list items so i can arrange them using flexbox commands

i used a flex-direction of row and flex:auto to spread them evenly, with text-align set to center

however i still had the issue of white space above, below and to the left of my list items

to investigate why this happening i used the inspect tool

i then reversed this in my main.css stylesheet

lastly i added an "active" class to identify which page was open and a hover for the nav bar

the changes were then reflected on my nav bar showing no whitespace or indent and a background colour added to the active page

i then pushed the changes, created a pull request and merged with my master branch (there were no conflicts)

to get the changes on my local machine i swapped to the master branch and used git pull
