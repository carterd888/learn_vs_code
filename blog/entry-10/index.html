What did i learn today?

i was able to complete the next section of the webpage without issue.

this consisted of a background image and a button in the center

however the final section i was determined to make a flexbox work

i set a container as the article and used the following flex commands

display: flex;
flex-direction: row;
align-items: stretch;
justify-content: space-between;
flex-wrap: wrap;

so that i got the contents to wrap at the correct time i manually added a width of the container to be 650px

i then used a margin-left to allign the flexbox in the center of the page. i started with a fixed px margin but changed this to a % value to make the page more responsive

margin-left: 30%;

the last part of the website to complete was the footer

i used the same method of a flexbox parent container justified to the center

i then made a class called iconBorder which would allow for uniform styling for each of the icon containers

.iconBorder {
margin: 5px;
align-items: center;
justify-content: center;
width: 40px;
height: 40px;
border-radius: 40px;
border: 1px solid lightgrey;
}

the border size and radius make it into a circle and ensure that the contents are aligned in the centre with a margin between them of 5px

with html the icons were generated from an online resource and styled using font-size and color
