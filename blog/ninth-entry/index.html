What did i learn today?

Another part of the homework for this week is to recreate a website using HTML and CSS

This is the site

i decided to break it down into sections and tackle the nav bar first

i used emmet to start the framework of the HTML

"header>nav>ul>(li>a)*6"

i added the text for the sections and a blank link "#"

using CSS i was able to style the nav bar to match the sample website however i struggled to get the logo to appear where i needed it to

i initially tried to make a container and use display:flex however the image seemed to resize to the entire container

the way i managed to resolve this issue was to float the image to the left with a max-width and max-height and use a margin to indent it

#karma-logo {
margin-left: 50px;
float: left;
height: auto;
width: auto;
max-height: 50px;
max-width: 50px;
}

and then i used display:inline for the list to position it to the right of the logo

nav ul li {
display: inline;
padding: 5px;
margin: 5px;
}
