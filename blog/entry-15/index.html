What did i learn today?

today i noticed a spelling mistake on my page title

while i was changing this i noticed that using this current indexing system not only are the names hard to read/spell they are also unordered within the blog directory

i proceeded to rename the pages using a numerical system starting with entry-1

this was quite time consuming as i had to rename each folder, rename the title in the head tag and h1 tag. then rename both the previous and next links to reflect this name change

the end result was a numerically ordered list which is much easier to read and impliment going forward
