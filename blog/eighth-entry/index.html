What did i learn today?

after discussing flexbox yesterday in class, we were given the task to complete a game using flexbox terminology

This is the site for the game

i was able to complete most of the levels without much difficulty, however i was stuck on the final level

i started by changing the flex-direction to a column in reverse order

i then added wrap-reverse so that first two yellow frogs appear on the first column

i then shortened this to flex-flow

next i aligned the content with space-between so that they were against the sides of the pond

to resolve this next step i thought i should select just the yellow frogs and manually adjust the order

this did not work as there was only the option to select the pond as a whole

eventually i came to the realisation that by centering the frogs they would line up differently on each side of the pond due to the number of frogs.

the final code i used was:

" flex-flow: column-reverse wrap-reverse;
align-content: space-between;
justify-content: center; "
