What did i learn today?

Today i learned about CSS Selectors. We were given the task to read up on the subject and attempt to complete a game to test our knowledge.

This is the Game

I found this a very useful tool to apply my knowledge.

One of the selectors i found challenging at first was the "*" selector used to select all, also known as "Universal Selector".

It was using this in combination with other selectors, specifically the placement of the "*" in the selctor where i struggled.

I first tried to use "plate*",

However this would be selecting the plates and not what is on the plates.

So i had to use the parent selector ">".

By combining the two as "plate>*" i was able to correctly selected every object that had the parent "plate".
