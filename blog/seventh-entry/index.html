What did i learn today?

today i wanted to improve the navigation between my blog pages

i first changed the links on the blog page to the dates of enteries instead of a numbered system first, second etc.

this will help to identify each blog as the page expands

my next idea was to include a way to move between the pages, based on which page you are on

i did some reasearch into this and it looked easier than i had anticipated using a similar method to my "nav bar"

because this would be a big change effecting many pages i decided to make a branch from my origin named "nav-scroll"

i used the terminal in visual studio code to do this as i had some previous experience

" git checkout -b nav-scroll "

i first reseached the buttons themselves, my initial idea would be to use directional arrows "<" and ">"

however i later decided this was not explicit enough in what they did so i added the text "Previous" and "Next"

i used this site to structure my CSS for the buttons

See site here

my first issue was that i had previous CSS to style a tags and as i placed this CSS further down my page it was overriden

to combat this i added a div tag around the navigation links with a class of next-previous and refered to this as the parent selector of my CSS styling for a tag

this allowed me to uniquely style these links and not affect any existing links

i edited the CSS to keep with the ongoing theme of my site with text and colours

once i was happy with the changes i had to push the changes and create a pull request

i used the terminal again to initiate this with

"git push --set-upstream origin nav-scroll"

then i managed the merge on github and deleted the branch

when i returned to my console i navigated back to the origin brach and used a pull command

"git checkout master"
"git pull"
